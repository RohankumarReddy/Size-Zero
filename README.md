## **Overview**

**Size-Zero** is an AI-powered fitness assistant that leverages computer vision to monitor user workouts, ensuring correct posture and movement.  
Using **MediaPipe**, it performs real-time pose estimation and rep counting. Post-workout analytics are enhanced with **Google Gemini** via **LangChain**, generating personalized reports that summarize performance and provide improvement suggestions.  

---

## **Tech Stack**

| Component | Technology Used | Description |
|------------|----------------|--------------|
| **Frontend** | Streamlit | Interactive user interface for capturing workouts and displaying results |
| **Computer Vision** | MediaPipe | Real-time human pose detection and landmark tracking |
| **AI/LLM Integration** | Google Gemini (via LangChain) | Generates workout summaries and recommendations |
| **Backend Logic** | Python | Core logic, video processing, and report generation |
| **Environment** | Anaconda / Google Colab/ VS code | Development and testing environment |

---

## **Features**

- Real-time pose detection using MediaPipe  
- Automatic rep counting and movement tracking  
- Post-session feedback generated by Gemini through LangChain  
- Streamlit-based interface for seamless user experience  
- Performance summary and form improvement insights
